# AJAX 协议

请求：`content-type: 'application/json'`

返回：
```json
{
    "resultCode": int,  // 0 代表没有错误，非 0 代表有错误
    "errorMessage": string,
    "data": object
}
```

## 注册
/register

请求：
```json
{
    "phone": string,
    "password": string
}
```

返回：
```json
{
    "userId": int,
}
```

## 获取作品列表
/getWorks

请求：无

返回：
```json
[{
    "workId": int,
    "name": string
}, ...]
```

## 加载作品
/loadWork

请求：
```json
{
    "workId": int
}
```

返回：
```json
[{
    "workId": int,
    "name": string,
    "materials": [
        {
            "materialId": 1,
            "type": "video",
            "url": "/ab/cd/efg.mp4",
        },
        ...
    ],
    其它项目数据
}, ...]
```

## 获取进度
/getProgress

请求：
```json
{
    "jobId": int
}
```

返回：
```json
{
    "progress": float   // 百分比
    "result": object  // 根据具体任务类型决定
}
```
